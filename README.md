# Grow Space Legal

Public legal pages for Grow Space internal integrations.

Deployed via Cloudflare Pages. Used by the Intuit Developer app assessment and any other platform that requires public EULA/Privacy URLs.

## Pages

- `/` — index
- `/eula` — End-User License Agreement
- `/privacy` — Privacy Policy

## Deployment

Push to `main` auto-deploys via Cloudflare Pages.
